# NEVERLASTING: Escape the Grand Selection

Survive the first stage of the Grand Selection in this fast-paced browser game inspired by the *Neverlasting* universe. Play as **Ichido**, evade danger, hide when necessary, and escape a deadly tournament where only a handful of participants survive.

Built as a **single-file HTML game** with no dependencies, no build tools, and support for desktop and mobile devices.

## 🎮 Play Online

**GitHub Pages:** https://sentakkuofficial.github.io/Neverlasting-Horror/

Replace the URL above after enabling GitHub Pages.

---

## ✨ Features

- Original story setting based on the *Neverlasting* world
- Play as Ichido during the Grand Selection tournament
- Mobile-friendly controls with an on-screen D-pad
- Desktop keyboard support
- Single-file architecture — easy to customize and deploy
- No frameworks, libraries, or installation required
- Runs entirely in the browser

---

## 🕹️ Controls

### Desktop

- **W / A / S / D** or **Arrow Keys** — Move
- **Space / E** — Hide or interact
- **P** — Pause game

### Mobile

- Use the on-screen directional pad
- Tap the **HIDE / USE** button to interact

## 📖 Story

Six hundred participants enter the **Grand Selection**.

Only a few will survive.

You are **Ichido**, a young contestant forced into a brutal tournament where alliances are fragile, danger lurks around every corner, and every decision matters.

Hide. Explore. Endure.

Escape the Grand Selection.

## 🚀 Getting Started

1. Clone this repository:

```bash
git clone https://github.com/yourusername/your-repository-name.git
```

2. Open the project folder:

```bash
cd your-repository-name
```

3. Launch the game:

- Double-click `index.html`, or
- Open it with a local server:

```bash
python -m http.server
```

4. Visit:

```text
http://localhost:8000
```

## 🌐 Deploy to GitHub Pages

1. Push your repository to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, select **Deploy from a branch**.
4. Choose:
   - Branch: `main`
   - Folder: `/root`
5. Save your changes.

Your game will be available at:

```text
https://yourusername.github.io/your-repository-name/
```
